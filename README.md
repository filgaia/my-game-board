# my-game-board
My Game Collection Board with Graphql

## Description

You will find here how to create an small games board

### How to run

- Enter `server` folder
- Run `npm run start`
- Access by http://localhost:9000/graphql The **Playground**
- Enter `client` folder
- Run `npm run start`
- Access by http://localhost:3000 The **UI**
